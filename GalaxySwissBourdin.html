Projet E5

Contexte

Le Laboratoire GSB (Galaxy Swiss bourdin) est une organisation pharmaceutique, c’est pour cela qu’elle a besoin d’entrer en contact avec les médecins. L’objectif du projet était donc de créer une application permettant d’afficher les départements des médecins ainsi que leur spécialité. Cela leur permettra aussi d’en ajouter des nouveaux et de supprimer ceux qui partent à la retraite.

Mon projet à été réalisé à partir de Visual Studio Code principalement accompagné du modèle de l'application web ASP.NET (.Net Framework).

J’ai crée une application BackEnd et FrontEnd en atteignant l’objectif de respecté les règles fixées par GSB. J’ai donc crée une base de données, qui permet au front d’avoir l’accès aux médecins et aux départements, ainsi que d’en crée ou d’en supprimer le contenu. C’est une application back-end respecant la norme RESTFULL tout en utilisant des données open data.
